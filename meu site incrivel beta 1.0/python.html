Função print

print("ola")

print(1) 

print("eu tenho",15,"anos.")

print é uma função(será explicado mais adiante), para usa-la, basta digitar "print" seguido de parenteses, dentro do parenteses coloque o que você quer que seja imprimido.

Se você quiser imprimir um texto, coloque ele entre aspas, se você quiser imprimir um número ou um bool(vou explciar em breve), não coloque as aspas.

Se você quiser misturar os dois casos, separe os diferente tipos de valores com virgulas.

No console, os exemplos aparecerão da seguinte forma respectivamente: ola, 1, eu tenho 15 anos.

Sequências de escape

print("Olá \nMundo!")
print("Nome\tIdade")
print("Este é um exemplo de barra invertida: \\")
print('Ele disse: \"Olá!\"')
print("\u2764")  # Coração

print("""
        olá 
        pessoas
        legais       
""")

Uma sequência de escape é uma combinação de caracteres que permite inserir caracteres especiais ou realizar ações específicas em uma string de texto. Existem diversos tipos de sequências de escape, vamos ver algumas delas.

\n é possivelmente a mais comum, sua função é realizar uma quebra de linha(pular para a proxima linha). \t Insere um caractere de tabulação(tab). \\ é usado para inserir uma barra invertida literal. \" serve para inserir aspas duplas(há também \'). \uxxxx é usada para representar caracteres Unicode, nesse caso, o xxxx é trocado por 2764, que representam um coração. É importante mencionar que se você colocar um espaço depois da sequência de escape, ela será interpretada pelo python.

Você pode escrever em multiplas linhas se usar 3 aspas(simples ou duplas), dessa foma não é necessário usar o \n para quebrar a linha.

Em python, existem 4 tipos primitivos, inteiros, float, string e booleano.

Os inteiros são todos os números que fazem parte do conjunto dos inteiros, e os floats são os números que fazem parte do conjunto dos racionais, para escrever um float, escreva ele com um ponto(não pode ser virgula), então no exemplo, "i" é um inteiro e "f" é um float.

Os strings são basicamente texto, e para escreve-los é necessário coloca-los entre aspas.

E os bool são só 2, True e False, para escreve-los, não use aspas e comece com letra maiuscula, além disso é possivel escreve-los usando operadores relacionais(sera explicado em breve).

Listas

lista = [1,2,3,4]
print(lista) #aparecerá 1,2,3,4 no console
print(lista[0]) #aparecerá 1 no console
print(lista[-1]) #aparecerá 4 no console
print(lista[-3]) #aparecerá 2 no console

print(lista[-10]) #aparecerá um erro
print(lista[0:2]) #aparecerá 1,2

OBS: O básico de listas já foi explicado anteriormente.

Para acessar uma lista, basta digitar seu nome, para acessar um item especifico da lista, você digita o nome da lista, um [], e o númeor do item da lista(os itens começam no 0).

Então se você quiser acessar o primeiro item de uma lista, coloque 0 nos [], o segundo item de lista, coloque 1, e assim sucessivamente.

Outra forma de acessar um item é usando um número negativo, então se você usar -1, será acessado o ultimo item da lista, -2, o penultimo item da lista, -3, antipenultimo, e assim por diante.

Se você tentar acessar um item que não existe, ocorrerá um erro.

Outra forma de chamar um item, é usando ":",da seguite forma: lista[começo:fim], no local indicado como "começo", coloque o indice do primeiro elemento que você quer chamar, e no local indicado "fim", coloque o elemento que você quer que pare de chamar, então se você colocar [0:3], os itens, 0,1 e 2 serão chamados.

As duas principais formas de adicionar itens em uma lista, são pelos métodos append, e insert, para usar um método, coloque um ".", o nome do método, e um ().

O método append simplismente coloca o que estiver dentro do () no final da lista, então no código exemplo, o "5" será colocado no final da lista.

Já o método insert, permite que você escolha onde colocar o item, usando a seguinte lógica: objeto.insert(localização, valor), no exemplo, o valor "0" será colocado no local 0(primeiro item da lista).

Para deletar um item de uma lista, ou uma lista toda, você usa a instrução del, como mostra o exemplo, para deletar um item especifico, basta dizer qual item você quer deletar, e para deletar a lista toda, basta colocar o nome da lista.

if, elif e else

if

x = 1
y = 2
if x == 1:
    print("x vale 1")
if y > 10:
    print("y é maior que 10")

Em python, para tomar decisões, usamos estruturas de controle de fluxo, if-elif-else. Veremos cada um separadamente, então vamos começar com if.

O comando "if" verifica se uma condição é verdadeira, se ela for, o código dentro do "if" é executado. No código ao lado, temos 2 exemplos, no primeiro, se "x" for igual a 1, a frase "x vale 1" será impressa, como a variavel x foi declarada com o valor 1, então a condição é verdadeira, e o código dentro do "if" será executado.

No segundo exemplo, o código está verificando se "y" é maior que 10, se sim, a frase "y é maior que 10" será impressa, como "y" vale 2, a condição é falsa, logo, nada acontecerá.

elif

x = 2
y = 3
if x == y:
    print("ola")
elif x < y:
    print("Jurandy False")
elif x != y:
    print("batatas são incriveis")

Elif serve para verificar se uma informação é verdadeira caso as anteriores sejam falsas, no exemplo, se "x" for igual a "y", será impresso "ola", se não, se o "x" for menor que o y, será impresso "Jurandy False", se não, se "x" for diferente de "y", será impresso "batatas são incriveis".

Como "x" vale 2 e "y" vale 3, a primeira condição é falsa, então o codigo irá testar a segunda condição(x < y), como 2 é menor que 3, a condição é verdadeira, e o codigo será executado, como uma das condições já se mostrou verdadeira, a terceira condição(x !=y) não será testada.

else

x = 1
y = 3
z = 9
if y > z:
    print("aaaaaa")
elif x < 0:
    print("bbbbbb")
else:
    print("cccccc")

Else serve para executar um codigo caso todas as outras condições sejam falsas, o codigo vai verificar o if, se ele for falso, vai verificar o(s) elifs, se todos forem falsos, será executado o else.

No exemplo, primeiro verifica se "y" é maior que "z", como é falso, será verificado se "x" é menor que "0", como é falso e é o único elif, será executado o else, ou seja, aparecerá "cccccc" no console.

if e elif

x = 1
y = 2
if x == 1:
    print("ola")
elif x > 1:
    print("ola2")
if y == 2:
    print("ola3")
#Aparecerá "ola" e "ola3" no console

Outra forma de usar os condicionais é se você quer que duas ou mais coisas aconteçam, por exemplo, se você quer verificar se duas coisas são verdadeiras, você pode usar 2 if, mas porque não usar o elif? Porque o elif só irá testar o código se a condição anterior for falsa.

No código de exemplo, estamos verificando se x é igual a 1, que é verdade, logo, o código do elif não vai acontecer, depois verificamos se y é igual a 2, que é verdade. Se em vez do segundo if, tivessemos usado um elif, o código teria parado no "x == 1", e como consequência, apenas "ola" teria sido impresso.

Mais exemplos

var1 = "ola"
var2 = 7
var3 = [1,3,5,7,9]
var4 = True
if var2 == var3[3]:
    print("Teste 1 é verdade")
else:
    print("Teste 1 não é verdade")
if var1 != "ola":
    print("aaaaa")
elif var2 > 5:
    print("bbbbb")
elif var4 == False:
    print("nada")
else:
    print("nada é verdade")

Nesse exemplo temos 4 variaveis, uma str("ola"), um int(7), uma lista([1,3,5,7,9]), e um bool(True). Na primeira condição, verificamos se a var2 é igual ao 4° item da var3(lembre-se que o primeiro item de uma lista tem indice 0), como var2 é 7, e o quarto item de var3 é 7, a condição é verdadeira, logo, "Teste 1 é verdade" será impresso no console.

Caso essa condição fosse falsa, "Teste 1 não é verdade" seria impresso no console, visto que ele está dentro de um bloco else, que é executado caso nenhuma das condições sejam verdades.

Na segunda condição, se var1 for diferente de "ola", a condição será verdade, como var1 é igual a "ola", é falso. Depois é verificado se var2 é maior que 5, como var2 é igual a 7, é verdade, logo, o código será executado, e "bbbbb" será impresso.

Caso a condição fosse falsa, seria verificado se var4 é igual a False, o que não é verdade, visto que var4 é igual a True. E por último, se nenhuma condição fosse verdadeira, o código print("nada é verdade") seria executado.

Outros operadores lógicos que podemos usar são o and, not e or. O "and" retornará True se as duas condições forem verdadeiras. No exemplo, é verificado se 1 é igual a 1, e se 2 é maior que 0, como as duas condições são verdadeiras, o resultado é True.

O "or" funciona de maneira similar, mas em vez de retornar True se ambas as condições forem verdadeiras, ele retorna True se pelo menos uma condição for verdade. E o "not" simplismente inverte o resultado da operação, se for True, vira False, e se for False, vira True.

Nos exemplos dados, os 3 "if" são verdadeiros.

Uma parte muito importante da programação, são as operações comuns, com elas, você pode atribuir valores a variaveis ou fazer contas. Também há os parênteses, que adicionam prioridade aquela conta.

A divisão(/) é diferente da divisão inteira(//), a primeira retorna um float, enquanto o segundo retorna um int. o resto(%) retorna o resto da divisão de um número por outro.

Todas essas operações seguem a mesmas prioridade que na matemática.

len() é uma função que retorna o tamanho de um objeto, se for uma lista por exemplo, ele vai reornar o número de valores que tem na lista, se for uma str vai retornar o númerod e caracteres.

range() é outra função, ela gera uma sequência de números de forma eficiente(eficiente significa conforme o nescessário). Ela recebe até 3 argumentos, da seguinte forma: range(início, fim, passo). O fim é obrigatório, enquanto o início e passo não. o fim indica onde irá parar de gerar, sendo o ultimo número gerado o antecessor do número indicado em fimm ou seja, se você colocar 10, o ultimo número gerado será 9. O início é onde começa a gerar, e o passo é A diferença entre cada número consecutivo na sequência, o valor padrão é 1.

Loops

for

for i in range(5):
    print(i)

frutas = ["maçã", "banana", "laranja"]
for fruta in frutas:
    print(fruta)

for i in range(1, 10, 2):
    print(i)

OBS: Os exeplos mostrados só mostram a função print dentro do bloco for, mas é possivel usar qualquer coisa dentro, inclusive outros loops.

loops são estruturas que permitem repetir um código várias vezes, até que uma condição ou até que todos os itens de uma sequência sejam processados. Existem 2 tipos de loops, for while. Começaremos falando do for.

for é uma estrutura de controle usada para iterar sobre uma sequência(como uma lista, tupla, string, ou qualquer objeto iterável), Com o for, você pode executar um bloco de código várias vezes, uma para cada item na sequência. O loop for cria uma variavel, que assume o valor de cada item da sequência. Depois que o for cria a variável que assume o valor de cada item da sequência, o bloco de código dentro do loop é executado para cada um desses itens. A cada iteração, a variável recebe um novo valor da sequência até que todos os itens tenham sido processados.

No primeiro exemplo,o range(5) gera uma sequência de números de 0 a 4, e o loop for itera sobre esses valores. No segundo, o for cria a variável fruta, que assume o valor de cada item da lista frutas, o loop irá imprimir cada fruta, uma de cada vez, até que a lista inteira tenha sido percorrida. E no terceiro exemplo, o range(1, 10, 2) gera números de 1 a 9, com incrementos de 2 (1, 3, 5, 7, 9).

while

while True:
    print("Olá, mundo!")

i = 0
while i < 5:
    print(i)
    i += 1

while e um loop que executa um bloco de código enquanto uma condição for verdadeira, isso pode causar alguns problemas, como loops infinitos, mas que não são muito difícis de resolver.

Para usar o while, basta colocar while e a condição, seguido de dois pontos. Se você colocar True como condição, o loop irá executar o bloco de código infinitamente. Então no exemplo, seria imprimido "Olá, mundo!" até o código ser encerrado. Se você colocar False como condição, o código não será executado.

No segundo exemplo, há uma variavel "i = 0" e um loop, que executará o código enquanto o valor de i for menor que 5. O código que é executado dentro do loop faz com que seja impresso o valor de i e aumente o valor de i em 1. Caso não houvesse o "i += 1", o loop seria infinito, já que i seria sempre menor que 5. Então como resultado do código, será impresso 0, 1, 2, 3, 4.

break, continue, pass

i = 0
while True:
    i += 1
    if i == 5:
        continue
    if i == 10:
        pass
    if i == 15:
        break
    print(i)

Break, continue e pass são instruções que controlam o fluxo de laços (loops) e blocos de código. Break serve para interromperer um loop. Continue serve para pular o loop atual e ir para o proximo, ou seja, tudo depois do continue ser usado será pulado, e o loop voltará para a primeira linha. E pass não faz nada, é uma instrução "vazia", usada como um espaço reservado.

No exemplo, se a variavel "i" for igual a 5, o loop voltará para a primeira linha, fazendo com que o 5 não seja imprimido. Se "i" for 10, então nada acontecerá e o código acontecerá normalmente, como se não houvesse um if, já que o pass não faz nada. Exemplo complexo

#Contador até o 50
num = 0
while True:
    num += 1
    #verificar se é divisivel por 10
    if num % 10 == 0 and num != 50:
        print("O número", num, "é divisivel por 10")
        continue
    #verificar se está na metade da contagem
    if num == 25:
        print("Chegamos na metade, ou seja, estamos no número", num)
        continue
    #verificar se contou até 32
    if num == 32:
        pass #nada aqui, só botei o pass para mostrar na prática com funciona
    #verificar se o contador chegou no 50
    if num == 50:
        print("Chegamos no final da contagem, contamos até o número", num)
        break
    print(num)

Nesse exmplo, temos um loop que conta até o número 50, como diz o comentário, e a declaração de uma variavel chamada num com o valor 0. Depois é iniciado um loop que se repetira até houver um break, pois como condição, é dado True. Na primeira linha do loop, o valor de num é aumentado em 1.

Depois há um if, ele verifica se num dividido por 10 tem resto 0(divisivel por 10) e se num é diferente de 50. Caso a condição seja verdadeira, será impresso o que está dentro da função print, e depois o loop reiniciará devido ao continue. No segundo if, o código verifica se num é igual a 25, se for, será executado o print, e depois um continue. No terceiro if, é verificado se num é igual a 32, se sim, nada acontece, pois dentro do bloco há um pass. E no ultimo if, se num for igual a 50, será executado o print dentro do bloco, e depois o loop será encerrado pelo break. Se nenhuma condição for verdadeira, num será impresso pelo print.

Como resultado do código, teremos todos os números de 1 a 50 impressos em linhas separadas, com exceção de 10, 20, 25, 30, 40 e 50. Esses números serão impressos de uma forma diferente de acordo com o print dentro do if.

Para chamar uma função, basta digitar o nome da função, parênteses

Funções são blocos de código reutilizáveis que realizam uma tarefa específica. Elas permitem que você organize seu código de forma modular e reutilize partes dele sem precisar duplicar o código.Para chamar uma função, basta digitar o nome da função, parênteses, e se houver algum parâmetro na função, digite ele dentro do parênteses.

Para criar uma função usa a palavra "def", seguido do nome da função, parênteses e dois pontos. Depois, dentro da função, escreva o que você quer que ela faça. No primeiro exemplo, criamos uma função chamada ola, que quando é chamada, imprime "ola".

Na segunda função, temos algo dentro do parênteses, é um parâmetro(Valores de entrada que a função pode usar), e no corpo da função esse parâmetro é somado a 1 e depois é usado o "return"(será visto em breve). Uma função pode ter quantos parâmetros for necessário, e para cada um, basta separa-los com virgula. Eles também podem ser predefinidos caso o usuario não insira eles, como no terceiro exemplo

Global

var1 = 1
ola = "ola"
def incrivel():
    global ola
    ola = "não ola"
    var1 = "batata"
    print(var1, ola)
incrivel()
print(var1, ola)

global é uma palavra-chave em python que serve para declarar dentro de um função se uma variavel existe fora dela, já que por padrão uma variavel declarada dentro da função, só existe dentro da função, então isso quer dizer que é possivel ter 2 variaveis com o mesmo nome? Não! A variavel declarada na função só pode ser usada dentro dela, então mesmo se ela tiver o mesmo nome de outra, fora da função ela terá o valor original.

No código de exemplo podemos ver isso com a variavel var1, dentro de "incrivel" é declarado uma variavel chamada var1, mas ela ja exite fora da função, então o que acontece quando tentamos imprimi-la? Se ela for impressa dentro da função, ela terá o valor "batata" e se for impressa fora da função, terá o valor 1.

Já a variavel ola, é definida como global dentro da função, então se ela for alterada dentro da função, também será alterada fora. Então como resultado do código, será impresso "batata não ola" na primeira linha, e 1 "não ola" na segunda.

def quinzena(dia):
    if dia >= 1 and dia <= 15:
        return "Primeira quinzena"
    elif dia >= 16 and dia <= 31:
        return "Segunda quinzena"
    else:
        return "Algo deu errado"
print(quinzena(12))

Nesse exemplo, estamos verificando qual é a quinzena do mês, e como parâmetro temos o dia. Se o dia for maior ou igual a 1 e menor ou igual a 15, a função retorna "Primeira quinzena", se não, se o dia for maior ou igual a 16 e menor ou igual a 31, retorna "Segunda quinzena", caso nenhuma condição for verdadeira, retorna "Algo deu errado"

E por ultimo imprimimos o resultado da função quinzena com 12 como parâmetro. Usamos o print pois sozinho o return não imprime o valor. Como resultado teremos "Primeira quinzena".
